# 📊 Alpha Vantage API 구조 설명

## Alpha Vantage API 종류

### 1️⃣ **기본 API (Core Stock APIs)**
- **TIME_SERIES_DAILY**: 일별 주가 Data
- **TIME_SERIES_INTRADAY**: 분봉 Data
- **GLOBAL_QUOTE**: 실시간 주가
- **SYMBOL_SEARCH**: 종목 검색

### 2️⃣ **Intelligence API** ⭐ (고급 서비스)
- **MARKET_STATUS**: 글로벌 시장 개장/폐장 Status
- **TOP_GAINERS_LOSERS**: 상위 상승/하락/거래량 종목
- **NEWS_SENTIMENT**: News 감정 Analysis
- **INSIDER_TRANSACTIONS**: 내부자 거래 Information
- **EARNINGS_CALL_TRANSCRIPT**: 실적 발표 대화록
- **ANALYTICS_SLIDING_WINDOW**: 고급 Analysis 도구

### 3️⃣ **Economic  지표 API**
- **REAL_GDP**: GDP Data
- **INFLATION**: 인플레이션 Data
- **UNEMPLOYMENT**: 실업률 Data

## 🧠 Intelligence API의 특징

### ✅ **고급 Analysis Features:**
- 시장 감정 Analysis
- 실시간 시장 Status
- 내부자 거래 추적
- News 기반 감정 점수

### ✅ **실시간 Data:**
- 글로벌 시장 현황
- 상위 변동 종목
- 시장 개장/폐장 Status

### ✅ **AI 기반 Analysis:**
- News 감정 Analysis
- 시장 심리 지수
- 트렌드 Analysis

## 🔑 API 키 Requirements

### **Free Tier (무료)**
- 기본 API: 25 requests/day
- Intelligence API: 제한적 접근

### **Premium Tier (유료)**
- 기본 API: 무제한
- Intelligence API: 전체 접근
- 실시간 Data
- 고급 Analysis 도구
